# ateaparty-reminder

## Set up

* prerequirements
    * [brew](https://brew.sh): for installing `hasura-cli`
    * [docker](https://docs.docker.com/docker-for-mac/install/)
    * [docker-compose](https://docs.docker.com/compose/install/)

### install hasura-cli

```bash
brew install hasura-cli
```

## Start a local server

```bash
docker-compose up
```
